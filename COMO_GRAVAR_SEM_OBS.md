# 🎥 COMO GRAVAR VÍDEO SEM OBS STUDIO

## 🎮 OPÇÃO 1: Xbox Game Bar (Windows 10/11) - RECOMENDADO

### Como Ativar:
1. **Windows + I** → Configurações
2. **Jogos** → **Xbox Game Bar**
3. Ativar "Gravar clipes de jogos..."

### Como Gravar:
- **Windows + G** → Abrir Xbox Game Bar
- **Windows + Alt + R** → Iniciar/Parar gravação
- **Windows + Alt + G** → Gravar últimos 30 segundos

### Onde Salva:
- **C:\Users\[SEU_USUARIO]\Videos\Captures**

---

## 📱 OPÇÃO 2: PowerPoint (Se tiver Office)

### Como Gravar:
1. Abrir **PowerPoint**
2. **Inserir** → **Gravação de Tela**
3. Selecionar área da tela
4. Clicar **Gravar**

---

## 🎬 OPÇÃO 3: Ferramenta Online Gratuita

### Loom (Gratuito):
1. Acesse: **loom.com**
2. Instalar extensão do Chrome
3. Clicar no ícone do Loom
4. Selecionar "Screen + Camera" ou "Screen Only"
5. Gravar

---

## 🚀 ROTEIRO SIMPLIFICADO PARA GRAVAÇÃO

### Preparação (30 segundos):
1. Fechar todas as abas do navegador
2. Organizar VS Code com projeto aberto
3. Executar: `GRAVAR_VIDEO_WINDOWS.bat`

### Gravação (2 minutos):
1. **Iniciar gravação** (Windows + Alt + R)
2. **0-30s:** Mostrar projeto no VS Code
3. **30s-1min:** Abrir dashboard.html, mostrar métricas
4. **1min-1:30min:** Clicar "Executar Automação", mostrar Edge abrindo
5. **1:30min-2min:** Mostrar produtos sendo cadastrados, dashboard atualizando
6. **Parar gravação** (Windows + Alt + R)
