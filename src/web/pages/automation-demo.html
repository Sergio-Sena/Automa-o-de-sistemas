🤖 Automação de Sistemas

Demonstração de automação web com cadastro de produtos

📋 O que faz

Esta automação demonstra um sistema completo de cadastro de produtos com as seguintes funcionalidades:

  • Login automático no sistema
  • Cadastro de 20 produtos do CSV
  • Preenchimento de 5 campos por produto
  • Logout automático ao final
  • Relatório de execução

⚡ Tecnologias

Projeto desenvolvido com tecnologias modernas:

  • Python + Selenium para automação
  • AWS Lambda para execução remota
  • HTML/CSS/JS para interface
  • S3 para hospedagem e dados
  • API Gateway para endpoints
