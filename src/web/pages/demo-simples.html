O que a Automação Faz

Sistema completo que automatiza o processo de cadastro de produtos em uma aplicação web.

  • Abre navegador automaticamente
  • Faz login no sistema
  • Cadastra 20 produtos do CSV
  • Preenche 5 campos por produto
  • Confirma cada cadastro
  • Faz logout ao final
🛠️

Tecnologias Utilizadas

Projeto desenvolvido com tecnologias modernas de automação e web.

  • Python para lógica principal
  • PyAutoGUI para automação desktop
  • HTML/CSS/JS para interface web
  • CSV para dados dos produtos
  • LocalStorage para persistência
📊

Resultados

A automação demonstra eficiência e precisão no cadastro de dados.

  • 20 produtos cadastrados automaticamente
  • 100% de precisão nos dados
  • Tempo de execução: ~2 minutos
  • Zero intervenção manual
  • Relatório de execução completo
